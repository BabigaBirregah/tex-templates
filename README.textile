h1. Presentation

@tex-templates@ is a collection of LaTeX styles for different usages.

h2. Slides

@tex-templates@ provides styles for creating slides with the Beamer tool.
- LE2I Laboratory located at the "Université de Technologie de Belfort-Montbéliard":http://www.utbm.fr
- The IRTES Research Institute of the "Université de Technologie de Belfort-Montbéliard":http://www.utbm.fr (DEPRECATED).

h2. PhD Theses

@tex-templates@ provides the official styles for writing a PhD thesis in the following Universities:
- "Université de Technologie de Belfort-Montbéliard":http://www.utbm.fr
- "Université de Franche-Comté":http://www.univ-fcomte.fr
- "Université de Bourgogne":http://www.u-bourgogne.fr

Examples for these PhD thesis styles may be found at: http://www.multiagent.fr/ThesisStyle

h2. French Research Supervision Accreditation

@tex-templates@ provides the unofficial styles for preparing a French Research Supervision Accredidation (In French: Habilitation à diriger des rerchers - HDR):
- Application document for "Université de Franche-Comté":http://www.univ-fcomte.fr
- HDR report for "Université de Franche-Comté":http://www.univ-fcomte.fr

h2. Masters

@tex-templates@ provides the unofficial styles for writing the Masters reports.

h2. Technical Reports

@tex-templates@ provides the styles for writing the technical reports of the Multiagent Group at the "Université de Technologie de Belfort-Montbéliard":http://www.utbm.fr.

- Presentation styles (for Beamer)

h2. Conference Papers

@tex-templates@ provides the styles for the following conferences:

h1. Installation

# For the reports, install the "@tex-upmethodology@ collection of packages":http://www.arakhne.org/tex-upmethodology 
# Copy the content of the current directory inside one of your @texmf@ directory, usually @/usr/share/texmf/tex/latex/tex-templates@ or @$HOME/.texmf@ on Unix operating systems.
# Update the LaTeX databases from the command line:

 $> mktexlsr
 $> update-updmap --quiet
